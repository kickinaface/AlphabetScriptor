Learning:

Learn how to use Alphabet Scriptor and the script


Alphabet:



Font Formats:

There are two formats of the script.

Traditional:


*A line format that is the first creation of this alphabet. This adaptation produces an effect that allows the learner to engage deeply with their intentions.


Print/Block Default:


*A block print format that removes the line and renders the alphabet characters only. This method is quick powerful and the first to be programmed.



Memorization:

*The picture in the top heading title above is the alphabet of this script and should be learned and memorized just as the learner understands the current alphabet.
This is to ensure the content is absorbed into the sub-conscious mind.
The ability to both read and write in this script or any script is imparitive before proceeding to computational typing or automated tools.
The traditional way of learning is the best way. In this particular case it is in alignment with previous writing styles that have already been acquired.
Once you are able to learn the script and read and write with it without reference material, you are ready to express with it and truly embrace this tool as well.



How to use: List


Add & Edit a Text Block:


-Click the: ADD button to bring up the modal.



-Insert your desired text into the dialog's input field.



-The new text block is shown.



-Click on the text block to edit the way the text is displayed.



-You can change the font style and the font size. Press Update when you are done.

Delete Text Block:


Press the Delete button to remove the text block.
!!PLEASE NOTE!! If you're having issues refocusing, click off the window and refocus back on the application after deleting a text block. This allows the input fields to work again. (Mostly observed on Windows but rare minor cases.)

Import & Export:


Click: Import to load in the data of a saved list workspace.
Click: Export to send your list workspace to a file in your home downloads folder. **The file names are different per workspace program. Be sure to import the correct workspace file.

Send to Type Writer:


Click: Send to Type Writer to send your content over to the Type Writer program. You can use this in various ways to build creative workflow styles.



How to use: Type Writer


Add Creative Text with Type Writer:


Type Writer is a program that can also be used to generate creative text. Alphabet Scriptor automatically saves your work so you don't have to worry about losing progress. Create unique writings that allow you to customize from another perspective.



Select Individual Characters:


-Click on each character or space and the cursor will blink indicating that your selected character is focused.
-From here, you can begin typing over the selected area.
-To delete a character or space, right click on it.



Change Font Size:


-Click on the: Font Size button.



-It will bring up a dialog showing where to input your numeric value for the font size.



Press the: Update button and the new font size will be set. Begin typing in order to see the new font size set.



Change Font Type:


-Click on the: Font button.



-It will bring up a dialog showing where to select either: Traditional or Block font type.



-Press the: Update button and the new font will be set. Begin typing in order to see the new font size set.



Export Workspace:


-Click on the: Export button.



-A popup message will appear, indicating that the workspace was exported to the download folder in the user's home directory.



Import Workspace:


-Click on the: Import button.



-It will bring up a dialog where you can browse and select the previously exported workspace. Select the file and the workspace is loaded in.


-After imported, press the: Esc key to enable typing again on the imported document.

New Workspace:


-Click on the: New button to create a new workspace. It will clear out the content on your current workspace. Just start typing again to see the text.



Edit Multiple Characters:


You can edit multple characters simultaneously with the Select button.

-First, click on the first character of the start of your selection. Then press the SELECT button.

-You will notice the cursor now blinks in highlight colors. Also, the Select button has changed color indicating the selection tool is active.

-Select the last character of the combined selection text. When you do, the selected combined text is highlighted. This means that those combined characters are selected to be modified together.

-Now that your text is selected, you can change the font size using the same button for single character edits.


-You can also change the font itself using the same method.


-Press the: DESELECT button to remove the highlighted combined selected text.


**Note! Selected text remains until you press: DESELECT. Also, you may use the selection tool multiple times if you need to edit multiple combined text.


Reset Type Writer:


-Sometimes while using the Type Writer for long periods of time and when the document page is very long, the Type Writer will need to be reset to regain speed and memory usage.
-You can either separate your pages so that it is less processor intensive, or you can SAVE and then press: RESET. This will clear out the previous memory and start the program over where you left off.
-Any time you feel the app is slow while using Type Writer, save and press reset and that should speed it up.
